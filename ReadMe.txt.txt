These are do. files that I created for running high frequency data checks on primary data. The programs that are written are customizable and I am working on re-writing them so they are applicable to other datasets. Also, the do. files are compatible with IPA's ipacheck. You can read more about that here: https://github.com/PovertyAction/high-frequency-checks/blob/master/ipacheck.pkg and https://data.poverty-action.org/data-quality/high-frequency-checks.html.
